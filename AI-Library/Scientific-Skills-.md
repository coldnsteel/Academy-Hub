## Install (Claude Code / Cursor / Codex)

```bash
# Add marketplace

# Install all skills
/plugin install scientific-skills@claude-scientific-skills

# Or clone directly
git clone https://github.com/K-Dense-AI/claude-scientific-skills.git
# Then copy skill folders into your .claude/skills/ or .codex/skills/ directory
```
